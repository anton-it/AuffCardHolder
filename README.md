Auff Card Holder 
Зудумывалось как приложение для хранение скидочных карт а стало приложением с заметками.
Цель приложения:
Изучение и практическое применение современных технологий Android разработки.

Использованные технологии:

    Язык - Kotlin
    Архитектура - MVVM
    UI - Jetpack Compose
    Хранение данных – Room
    Асинхронная работа - Kotlin Coroutins
    Dependency injection - Hilt
 

Использованные API:
Не используется
