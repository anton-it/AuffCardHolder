# Auff Card Holder

Задумывалось как приложение для хранение скидочных карт а столо - приложение заметки с вохможностью добавления фото к заменткам. 

## Возможности
 - Создание, редактирование и удаление заметок всего в пару касаний
 - Возможность закреплять приоритетные заметки в верхней части списка
 - Поддержка добавления изображений к текстовым заметкам
 - Функция поиска по содержимому на главном экране. Поиск ведется по общему списку и закрепленным заметкам.
 - 

## Использованные технологии:
1. Язык - Kotlin
2. Архитектура - MVVM
3. UI - Jetpack Compose
4. Хранение данных – Room
5. Асинхронная работа - Kotlin Coroutins
