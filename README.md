# Auff Сфкв Рщдвук

## Задумывалось как приложение для хранение скидочных карт а столо приложение с заметками. 

## Цель приложения: 
 Изучение и практическое применение современных технологий Android разработки.

## Использованные технологии:
1. Язык - Kotlin
2. Архитектура - MVVM
3. UI - Jetpack Compose
4. Хранение данных – Room
5. Асинхронная работа - Kotlin Coroutins
